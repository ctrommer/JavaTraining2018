Was sollte der Benutzer "hören", wenn jemand ein Optional verwendet?
Achtung, der Wert kann leer sein.

Wann sollte man Optional verwenden?
Wenn bei einer Methode der Rückgabewert "kein Ergebnis" möglich ist, aber null als Rückgabewert wahrscheinlich zu Fehlern führt.   

Sollte man Optional als generelles "Vielleicht ist der Wert vorhanden" verwenden?

Sollte man Optional verwenden, wenn eine Methode eine Liste von Werten zurückgibt?
Nein. 

Was sollte man statt Optional verwenden, wenn eine Methode eine Liste von Werten zurückgibt?
Eine leere liste.

Sollte man Optional für Felder ( Member ) verwenden?
Nein.

Sollte man Optional für einen optionalen Parameter einer Methode verwenden?
Nein.

Sollte man Optional in Collection Klassen verwenden?
Nein

Was ist zu myOptional.get() zu sagen?
Man sollte es besser gar nicht verwenden. 

Was sollte man machen, wenn man myOptional.get verwendet?
Es absichern mit:
if(soundcard.isPresent()){
  System.out.println(soundcard.get());
}

Was sollte man statt Optional.get verwenden?
Optional.orElse oder ifPresent
